Study Planner Web App
Project Description

The Study Planner Web App is a simple productivity tool designed to help students organize their study schedules and stay on top of assignments, exams, and tasks. The goal of this project is to create an easy-to-use website where users can plan their study sessions, track tasks, and manage their time more effectively.

Planned Features

Assignment deadline tracker: Users can add, edit, and delete study tasks or assignments.

Tracked study sessions: Users can organize tasks by date or time to plan study sessions.

Task Completion Tracking: Users can mark tasks as completed to track their progress.

Pomodoro Timer: Timer to help study.

Flash cards: Feature for students to write flash cards

Tech Stack

HTML – Structure and layout of the website

CSS – Styling and responsive design

JavaScript – Interactive functionality for task management and scheduling

Git & GitHub – Version control and project hosting
